About RunnersBlog

BUILT FOR
RUNNERS
BY RUNNERS

RunnersBlog is a community platform where every runner — from first-timers to ultramarathon veterans — can share their journey, inspire others, and find their tribe.

Our Mission

Every Mile Has a Story Worth Telling

Running is one of the most personal pursuits a human can undertake. Every run — whether it's a 5-minute jog or a 100-mile ultra — carries with it a unique story of determination, struggle, and triumph.

We built RunnersBlog because we believe those stories deserve to be told. Not just for the person who ran them, but for every runner out there searching for motivation, guidance, or simply the reassurance that they are not alone.

Our platform is free, welcoming, and built from the ground up to serve the running community — whether you run for medals, for health, or just for the sheer joy of it.

Technical Architecture

Built With Modern Tech

RunnersBlog uses a 3-tier architecture for scalability, maintainability, and performance.

🎨

FRONTEND

• HTML5 + CSS3 + Vanilla JS • Responsive mobile-first design • Rich text editor (contenteditable) • JWT auth stored in localStorage • Fetch API for REST calls
⚙️

BACKEND

• Node.js + Express.js • RESTful API architecture • JWT authentication (bcrypt) • MVC pattern (routes/controllers) • CORS enabled for security
🗄️

DATABASE

• PostgreSQL (pg driver) • 4 relational tables • Foreign key constraints • Optimized indexes • Connection pooling
